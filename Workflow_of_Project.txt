# Process of how to work on this Project 

1 : Env Setup
    - First create a backend folder which have config.py file having pydantic BaseSetting implemented
    - Create a .env file with the Credentials which can be used in the program

2 : Backend Creation
    - create a models folder having db_models.py which have the model for our database first start with user model 
    
    - create a utils folder having __init__.py and auth.py which have all the authentication basic function code
        -> hash_password 
        -> verify_password
        -> create_access_tokens
        -> create_refresh_tokens
        -> decode_tokens
        -> get_current_user

    - Create Router folder in backend and make a Auth.py file in it having below function 
        -> first of all go to models then db_models make pydantic request response schemas like
            - UserCreate :- For Signup 
            - UsageStat  :- For check how much user is using our services
            - UserLogin  :- For Login
            - TokenResponse :- For Response of JWT tokens 
            - UserOut :- for getting user 
        -> now in the Auth.py file make the below App router 
            - /register 
            - /login 
            - /refresh 
            - /me 
        -> now make a rate_limiter file in middleware folder which should be in the backend to set the limit of request given by the user for not overwhelming the app
